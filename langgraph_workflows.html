Onelastdance LangGraph Workflow Diagrams

Easy visual explanation of each Python file and the general LangGraph flow.

Overall LangGraph Pattern

flowchart TD
    A["1. Define State
TypedDict"] --> B["2. Create Nodes
Python functions"]
    B --> C["3. Build Graph
StateGraph(State)"]
    C --> D["4. Add Nodes
graph.add_node(...)"]
    D --> E["5. Connect Flow
add_edge / add_conditional_edges"]
    E --> F["6. Compile
app = graph.compile()"]
    F --> G["7. Run
app.invoke(initial_state)"]
    G --> H["8. Final State
result dictionary"]

    classDef setup fill:#e8f3ff,stroke:#1b6aa8,color:#0b2740
    classDef run fill:#eaf7ea,stroke:#2f7d32,color:#123b14
    class A,B,C,D,E,F setup
    class G,H run

iterative_tools.py - Writer, Search Tool, Reviewer Loop

flowchart TD
    START(["START"]) --> INIT["Initial State
topic, messages, draft,
review_feedback, is_approved, attempt"]
    INIT --> W["writer_node
Write or rewrite post
attempt = attempt + 1"]

    W --> TC{"Did writer request
TavilySearch tool?"}
    TC -->|"Yes"| TOOL["ToolNode
Run Tavily search"]
    TOOL --> W

    TC -->|"No"| EX["extract_draft_node
Save last AI message as draft"]
    EX --> REV["reviewer_node
Approve or reject draft"]

    REV --> STOP{"Approved OR
attempt >= 3?"}
    STOP -->|"No"| FB["Save review_feedback
Loop back to writer"]
    FB --> W
    STOP -->|"Yes"| END(["END
Print final post"])

    class INIT,FB state

    class TC,STOP decision
